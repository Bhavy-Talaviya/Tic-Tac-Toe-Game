## 📖 About the Project

Tic-Tac-Toe is a classic two-player game where players take turns marking spaces in a 3×3 grid.  
The objective is to be the first to place three of your marks in a horizontal, vertical, or diagonal row.

This project focuses on:
- Writing clean JavaScript logic
- Handling user interactions
- Managing game state
- Building a responsive UI

---

## 🚀 Features

- ✅ Two-player mode (Player X vs Player O)
- 🎯 Automatic win detection
- 🤝 Draw detection
- 🔄 Restart / Reset functionality
- ⚡ Fast and lightweight
- 🎨 Clean and minimal UI
- 📱 Responsive design (works on mobile & desktop)
